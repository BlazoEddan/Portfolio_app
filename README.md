# 💼 Streamlit Portfolio App

## 📌 Overview

This project is a **Multipage Portfolio Web Application** built using the Streamlit framework in Python.
It showcases my personal profile, technical skills, projects, and contact information in an interactive and user-friendly interface.

---

## 🚀 Features

* 🏠 Home Page – Introduction and overview
* 👤 About Page – Personal background and goals
* 🛠️ Skills Page – Technical skills and tools
* 📂 Projects Page – List of my projects
* 📞 Contact Page – Contact form and social links
